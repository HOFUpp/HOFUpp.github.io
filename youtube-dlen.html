youtube-dl

youtube-dl is an open-source and multiplatform command-line program to download video(and audio) from youtube and some other sites.

why should I use it?

youtube-dl youtube-dl is the best tool for downloading video from most web pages. A problem for some user could be that the program is usable only from the terminal);

if someone doesn't want to learn how to use the terminal there is another project called youtube-dl-gui that adds a graphical interface to youtube-dl. You can download it here

how to use it

all youtube-dl commands have the same syntax

$ youtube-dl [OPTIONS] URL [URLâ€¦]

for example:

$ youtube-dl https://www.youtube.com/watch?v=dQw4w9WgXcQ

to see all possible options you can consult the documentation here or use the following command:

$ youtube-dl -h
